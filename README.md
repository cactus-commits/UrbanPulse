# Bokoll

Detta repository innehåller projektet Bokoll, en datadriven visualisering med fokus på att utforska och kommunicera områden i Stockholm genom interaktiva dashboards.

Projektet syftar till att omvandla komplex data till tydliga visuella insikter som stödjer analys, förståelse och beslutsfattande.

---

## Översikt

Projektets kärna är en Power BI-applikation där data presenteras genom noggrant utformade visualiseringar med fokus på:

- tydlighet  
- estetik  
- interaktivitet  
- användarupplevelse  

---

## Visualisering och design

Det visuella uttrycket är centralt i Bokoll. Dashboarden är utformad för att:

- lyfta fram mönster och samband utan att överbelasta användaren  
- använda färg, kontrast och layout för att styra uppmärksamhet  
- möjliggöra intuitiv interaktion och utforskning  
- kombinera flera datakällor i en enhetlig visuell struktur  

Designen strävar efter att balansera analys och presentation, där varje komponent har ett tydligt syfte.
